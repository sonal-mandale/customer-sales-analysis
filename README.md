# 📊 Customer Sales Analysis Dashboard

### 🛠️ Tools Used

* SQL (MySQL)
* Power BI
* Microsoft Excel

## Project Overview

This project analyzes customer sales data to identify purchasing patterns, high-performing product categories, and revenue trends. Using SQL for data preparation and Power BI for visualization, the dashboard provides interactive insights that support data-driven business decisions.

## Key Insights

* Analyzed sales performance by gender and age group.
* Identified top-selling product categories.
* Tracked monthly sales trends and revenue growth.
* Created KPI cards for Total Revenue, Total Orders, and Average Order Value.
* Designed interactive filters for easy data exploration.
